# GitHub Pulse Dashboard - Agent Instructions

This is a short OJT application challenge project for Jazhem Hamid.

## Goal
Build a small but useful dashboard that pulls from a public API and displays helpful insights.

## Tech Stack
- Next.js App Router
- TypeScript
- Tailwind CSS
- GitHub public API
- No database
- No auth
- No API keys

## Rules
- Keep it simple and finishable in 60 to 90 minutes.
- Make it look polished but not overbuilt.
- Use clean reusable components.
- Add loading, empty, and error states.
- Do not add unnecessary dependencies.
- Do not expose secrets.
- Run npm run build and npm run lint after changes.
